This add on for PropertyHive allows you to manage property maintenance jobs.

== Installation ==

= Manual installation =

The manual installation method involves downloading the Property Hive Property Maintenance Add-on plugin and uploading it to your webserver via your favourite FTP application. The WordPress codex contains [instructions on how to do this here](http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).

= Updating =

Updating should work like a charm; as always though, ensure you backup your site just in case.

== Changelog ==

= 1.0.5 =
* Added support for reCAPTCHA v3
* Added support for hCaptcha
* Declared support for WordPress 5.8.1

= 1.0.4 =
* Change notes to be in own tab like other post types in Property Hive, as opposed to being in sidebar
* Change how status changes are stored in notes to match how status changes for other post types are stored

= 1.0.3 =
* Added new [report_maintenance_issue_form] shortcode for 'Report Issue' form to be placed on frontend. Reported jobs will eb emailed through and will go in as a maintenance job with a status of 'Pending - Unacknowledged'
* Added status filter to maintenance jobs list
* Added sanitisation
* Declared support for WordPress 5.7

= 1.0.2 =
* Added ability to add notes to maintenance jobs
* Job status changes are recorded and displayed in history
* Landlords can now view maintenance jobs on their properties from within their account
* Declared support for WordPress 4.8.1

= 1.0.1 =
* Updated main admin maintenance jobs list to include extra columns/information
