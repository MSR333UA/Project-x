== Description ==

This add on for Property Hive adds what3words integration to the property record and a property page on your website.

== Installation ==

= Manual installation =

The manual installation method involves downloading the Property Hive what3words Add-on plugin and uploading it to your webserver via your favourite FTP application. The WordPress codex contains [instructions on how to do this here](http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).

Once installed and activated, you can access the settings for this add on by navigating to 'PropertyHive > Settings > what3words' from within Wordpress.

= Updating =

Updating should work like a charm; as always though, ensure you backup your site just in case.
