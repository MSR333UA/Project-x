== Installation ==

= Manual installation =

The manual installation method involves downloading the Property Hive White Label plugin and uploading it to your webserver via your favourite FTP application. The WordPress codex contains [instructions on how to do this here](http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).

Once installed and activated, you can configure the add on by navigating to 'Property Hive > White Label' from within Wordpress.

= Updating =

Updating should work like a charm; as always though, ensure you backup your site just in case.
