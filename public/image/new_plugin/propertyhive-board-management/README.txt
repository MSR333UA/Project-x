== Description ==

This add on for PropertyHive allows you to manage property boards and board contractors.

== Installation ==

= Manual installation =

The manual installation method involves downloading the PropertyHive Board Management Add-on plugin and uploading it to your webserver via your favourite FTP application. The WordPress codex contains [instructions on how to do this here](http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).

Once installed and activated, a new 'Board' tab will appear when adding and editing properties. You can access the settings  for this add on by navigating to 'PropertyHive > Settings > Board Management' from within Wordpress.

= Updating =

Updating should work like a charm; as always though, ensure you backup your site just in case.
