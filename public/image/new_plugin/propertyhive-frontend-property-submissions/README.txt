== Description ==

This add on for Property Hive allows users to submit properties via the frontend of the website.

== Installation ==

= Manual installation =

The manual installation method involves downloading the Property Hive Frontend Property Submissions Add-on plugin and uploading it to your webserver via your favourite FTP application. The WordPress codex contains [instructions on how to do this here](http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).

Once installed and activated, you can access the settings this add on by navigating to 'Property Hive > Settings > Frontend Submissions' from within Wordpress.

= Updating =

Updating should work like a charm; as always though, ensure you backup your site just in case.
