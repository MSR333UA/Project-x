This add on for Property Hive provides the ability to set automatic rotation of property photos.

== Description ==

Changing the primary property photo is a tried and tested way of increasing interest and views on a property. This add on for Property Hive provides the ability to set automatic rotation of property photos.

== Installation ==

= Manual installation =

The manual installation method involves downloading the Property Hive Rotate Property Photos Add-on plugin and uploading it to your webserver via your favourite FTP application. The WordPress codex contains [instructions on how to do this here](http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).

Once installed and activated, you can access the settings for this add on by navigating to 'Property Hive > Settings > Rotate Photos' from within WordPress.
