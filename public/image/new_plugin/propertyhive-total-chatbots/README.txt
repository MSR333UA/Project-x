== Description ==

This add on for Property Hive adds the ability to display a chatbot, powered by Total Chatbots, on your frontend site.

Please enter referral code '3GF8VG' when signing up to Total Chatbots or mention you're using Property Hive.

== Installation ==

= Manual installation =

The manual installation method involves downloading the Property Hive Total Chatbots Add-on plugin and uploading it to your webserver via your favourite FTP application. The WordPress codex contains [instructions on how to do this here](http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).

= Updating =

Updating should work like a charm; as always though, ensure you backup your site just in case.
