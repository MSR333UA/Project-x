This add on for Property Hive outputs a Moving Cost Calculator

== Description ==

This add on for Property Hive outputs a Moving Cost Calculator. Simply add the shortcode [moving_cost_calculator] where you'd like the calculator to appear.

== Installation ==

= Manual installation =

The manual installation method involves downloading the Property Hive Moving Cost Calculator plugin and uploading it to your webserver via your favourite FTP application. The WordPress codex contains [instructions on how to do this here](http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).

Once installed and activated, you can configure the add on by navigating to 'Property Hive > Moving Cost Calculator' from within Wordpress.

= Updating =

Updating should work like a charm; as always though, ensure you backup your site just in case.
